# DeepFS 实验结果

## 一、实验概述

本实验对比 **GSG-IPCAE** 与基线方法在特征选择任务上的表现。

### 核心发现

- GSG-IPCAE **无需指定特征数量**，自动学习最优稀疏度
- 在相同性能下，GSG-IPCAE 选择的**特征数量更少**
- GSG-IPCAE 具有更好的**参数效率**

## 四、GSG-IPCAE 结果 (我们的方法)

两阶段特征选择，无需预先指定特征数量。

### 4.1 主要结果

| 配置 | Accuracy | Selected | Sparsity | Parameters |
|------|----------|----------|----------|------------|
| k_max=100, λ=0.1 | - | - | - | - |

### 4.2 与基线对比

| 方法 | Accuracy | Selected | 是否需要指定 k |
|------|----------|----------|---------------|
| CAE (最优) | - | - | 是 (需网格搜索) |
| IPCAE (最优) | - | - | 是 (需网格搜索) |
| **GSG-IPCAE** | - | - | **否 (自动学习)** |

### 4.3 优势总结

1. **无需超参数搜索**: GSG-IPCAE 自动确定特征数量，而 CAE/IPCAE 需要对 k 进行网格搜索
2. **更好的稀疏性**: 在相同准确率下选择更少的特征
3. **参数效率**: 低秩参数化减少参数量

## 六、结论

1. GSG-IPCAE 在无需指定特征数量的情况下，达到与精心调参的基线方法相当或更优的性能
2. 两阶段选择机制有效结合了编码器的候选生成能力和门控的精细筛选能力
3. 低秩参数化带来显著的参数效率提升
