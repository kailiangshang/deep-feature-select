# DeepFS Wiki

欢迎来到 DeepFS (Deep Feature Selection) 项目文档！

## 项目简介

DeepFS 是一个灵活、模块化的深度特征选择库，实现了多种最先进的可微特征选择算法。它专为高维数据（如基因表达数据）的特征选择而设计。

## 核心特性

- **模块化设计**: 门控(Gate)、编码器(Encoder)、选择器(Selector)可自由组合
- **多种算法**: 实现 4 种门控算法 + 2 种编码器算法 + 8 种组合方法
- **易于扩展**: 清晰的基类设计，方便实现新算法
- **生产就绪**: 完整的类型注解、文档字符串

## 核心算法: GSG-IPCAE

我们提出的核心算法是 **嵌入式 Gumbel-Softmax 门控 (GSG) + 间接参数化混凝土自编码器 (IPCAE)** 的组合，实现两阶段特征选择：

1. **第一阶段 (IPCAE)**: 从原始 D 维特征中选择 k_max 个候选特征
2. **第二阶段 (GSG)**: 从 k_max 个候选中进一步筛选最优特征

### 优势

| 特性 | 传统方法 | GSG-IPCAE |
|-----|---------|-----------|
| 参数量 | O(D) 或 O(D²) | O(D×d) 低秩参数化 |
| 特征数量 | 固定或不稳定 | 两阶段精确控制 |
| 稀疏性 | 需要 L1 正则 | 天然稀疏 + L0 正则 |
| 可微性 | 部分可微 | 完全可微 |
